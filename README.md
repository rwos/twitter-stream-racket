twitter-stream-racket
=====================

A twitter stream server (JSON over HTTP) written in Racket.

To run the (very basic) tests, do `make test`.

To start it, do

    make twitter-streamd
    ./twitter-streamd

It listens on port 8080 by default (see start.rkt).
